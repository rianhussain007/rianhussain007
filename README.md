## 🚀 Featured Project — MediMind

<div align="center">

<a href="https://github.com/rianhussain007/MediMind">
<img src="https://img.shields.io/badge/💊_MediMind-Personal_Medicine_Reminder_App-1565C0?style=for-the-badge&labelColor=0D47A1" />

</div>

> A full-featured Android app that helps users **never miss a medicine dose** again.

| Feature | Description |
|---|---|
| 🔐 **Firebase Auth** | Secure login & registration per user |
| 💊 **Medicine Manager** | Add medicines with dosage, frequency & notes |
| ⏰ **Smart Reminders** | Daily notifications via AlarmManager |
| 📋 **Medicine List** | View, toggle active/inactive, delete entries |
| 📊 **Dashboard** | Live count of active reminders |
